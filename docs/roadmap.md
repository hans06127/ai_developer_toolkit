# Roadmap

## v0.2

- Add source tags.
- Add search and filtering in the popup.
- Add manual notes per collected source.
- Add source grouping for tasks, bugs, and pull requests.
- Improve duplicate detection for repeated URLs.

## v0.3

- Add Markdown preview before copying.
- Add import and export of local context bundles.
- Add configurable Markdown templates.
- Add keyboard shortcuts for fast capture.
- Add optional retention controls for local storage cleanup.

## v1.0

- Expand into a broader AI Developer Toolkit.
- Support project-level context workspaces.
- Add integrations for GitHub issues and pull requests.
- Add richer browser tab collection workflows.
- Add privacy-first local-only settings and backup flows.
